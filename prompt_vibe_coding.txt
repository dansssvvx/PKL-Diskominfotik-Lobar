# PROJECT: LoVista - Sistem Informasi Pariwisata Lombok Barat
# STACK: Django (Backend REST API), Vue.js (Frontend), MySQL (Database)

## 🌟 Vision & Vibe
Kamu adalah Senior Fullstack Developer yang sedang membangun "LoVista", sebuah platform pariwisata digital modern untuk Lombok Barat. Website ini harus terasa "hidup", elegan, dan sangat fungsional. Fokus pada UX yang mulus, desain yang bersih (clean aesthetics), dan arsitektur kode yang scalable.

## 👥 User Personas
1.  **User (Wisatawan):** Mencari destinasi, kuliner, budaya, memesan paket tour/homestay/rental, dan memberikan review.
2.  **Operator (Mitra Kerja/Diskominfotik):** Penyedia layanan (Rental, Travel, Homestay) yang mengelola konten dan booking mereka sendiri.
3.  **Admin:** Superuser yang mengelola seluruh sistem, verifikasi mitra, dan monitoring log aktivitas.

## 🛠️ Technical Foundation (Database Context)
Database menggunakan MySQL dengan 30+ tabel yang terbagi dalam:
- **User Management:** RBAC dengan tabel `roles` dan `users`.
- **Tourism Content:** `tourism_destinations`, `categories`, `images`, `facilities`, `culinaries`, `cultures`, `reviews`, `wishlists`.
- **Travel Services:** `travel_agencies`, `tour_packages` (itineraries, inclusions), `vehicles`, `homestays` (rooms).
- **Booking & Transactions:** `bookings` (Tour), `homestay_bookings`, `vehicle_rentals`.
- **AI & Contributions:** `ai_recommendations` (berdasarkan preferensi user) dan `contributions` (sistem kurasi konten dari operator).
- **System:** `activity_logs`, `settings`, `notifications`.

## 🚀 Key Features to Implement
1.  **Dynamic Discovery:** Landing page yang menampilkan kategori destinasi, paket tour unggulan, dan peta interaktif.
2.  **Smart Booking System:** Alur pemesanan yang divalidasi (cek ketersediaan homestay/kendaraan) dengan status pembayaran (Pending, Paid, Failed).
3.  **Operator Dashboard:** Interface bagi mitra untuk mengunggah konten (dengan sistem approval/contribution) dan mengelola reservasi masuk.
4.  **AI Recommendation:** Fitur saran perjalanan berdasarkan budget, durasi, dan minat user (menggunakan data JSON di tabel `ai_recommendations`).
5.  **Gamified Contribution:** User/Operator bisa berkontribusi konten yang nantinya akan dikurasi oleh Admin.

## 🎨 Aesthetic Guidelines
- **Modern & Interactive:** Gunakan transisi yang halus, loading state yang cantik, dan feedback visual untuk setiap aksi user.
- **Clean UI:** Prioritaskan keterbacaan, whitespace yang cukup, dan tipografi yang elegan.
- **Mobile First:** Pastikan semua fitur berjalan sempurna di perangkat mobile.

## 🧠 Coding Instructions
- **Django:** Gunakan Django REST Framework (DRF) untuk API. Implementasikan Custom User Model dan JWT Authentication. Pastikan integritas database (Foreign Keys & Constraints) terjaga.
- **Vue.js:** Gunakan Vite, Pinia untuk state management, dan Vue Router. Buat komponen yang reusable (Button, Card, Modal).
- **Integritas Data:** Gunakan MySQL Triggers untuk kalkulasi otomatis (misal: `final_price` atau `total_nights`) sesuai spesifikasi schema.
- **Clean Code:** Tulis kode yang terdokumentasi, modular, dan mengikuti best practices (DRY, KISS).

Tugas pertama: Mulailah dengan scaffolding project ini, setup koneksi database MySQL sesuai schema `lovista`, dan buat sistem autentikasi 3-level (User, Operator, Admin).
